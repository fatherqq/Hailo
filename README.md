# Hailo-ollama
- Hailo Local LLM Add-on

## Enter the Add‑on Store
In the Home Assistant sidebar, click Settings. Then, in the Settings page, find and click Add‑ons, and finally click the Add‑on Store button at the bottom right.
## Open Repository Management
In the top right corner of the Add‑on Store page, click the ⋮ (three‑dot menu). From the dropdown, select Repositories.
## Add the Repository URL
In the pop‑up dialog, paste the following repository URL into the input field:
https://github.com/fatherqq/Hailo
Then click the Add button.
## Finish Adding
Once added successfully, all add‑ons from this repository will appear in your Add‑on Store. You can install them just like official add‑ons by clicking the install button.
N## otes
This method works for Home Assistant OS and any installation type that includes Supervisor.
